# wlspotifyOverlay

A lightweight spotify overlay for your desktop. Made in qt6/C++. Displays the current song title, artits and album cover fetched from spotify via DBus.

### Features:
- Real-time Metadata
- Minimalist Design
- Mouse controls:
    - **Left Click**: Play / Pause 
    - **CTRL + Scroll Down/Up**: Adjuste spotify volume
    - **Right Click**: Jump to next song

**Requeriments:** `qt6`

### Installation
```bash 
git clone https://github.com/backendbtw/zSpotifyOverlay.git
cd zSpotifyOverlay
mkdir build && cd build 
cmake .. && make
```
